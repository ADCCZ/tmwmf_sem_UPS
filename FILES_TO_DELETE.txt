## UPLOAD NA SERVER

### Varianta A: Upload na home.zcu.cz
1. Vytvoř ZIP archiv:
   zip -r pexeso_projekt.zip . -x ".git/*"

2. Upload přes SCP:
   scp pexeso_projekt.zip login@home.zcu.cz:~/public_html/ups/sem/

3. Na serveru rozbal:
   ssh login@home.zcu.cz
   cd ~/public_html/ups/sem/
   unzip pexeso_projekt.zip

### Varianta B: Push na GitHub
1. Vyčisti projekt (ale PONECH .git/!)
   ./CLEANUP_BEFORE_UPLOAD.sh

2. Commit a push:
   git add .
   git commit -m "Clean build artifacts before release"
   git push origin main
